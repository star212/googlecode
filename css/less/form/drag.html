drags演示，请用手机
使用方法： 
                var v = new drag({
                    hor : "lock",
                    el: "#v",
                    onDrag: function() {
                        this.css("opacity","0.5");
                    },
                    onStop: function() {
                        this.css("opacity","");
                    }
                });
参数类型默认值描述
elstring.drag元素的选择器
horstringfalse横向是否锁定，false为可以横向拖动,lock为锁定
verstringfalse纵向是否锁定，false为可以纵向拖动
startXnumber0起始横坐标值
startYnumber0起始纵坐标值
onDragfunction空拖动时回调函数，x为横坐标，y为纵坐标
onStopfunction空拖动停下时回调函数，x为横坐标，y为纵坐标
leftLimitnumber0左边界，一般相对于父容器，0为最左边。对x做限制
topLimitnumber0上边界，一般相对于父容器，0为最上边。对y做限制
rightLimitnumber父容器和元素之间宽度右边界
bottomLimitnumber父容器和元素之间的高度下边界
